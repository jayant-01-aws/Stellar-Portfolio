Burnout Predictor

An end-to-end Machine Learning solution designed to identify early signs of employee exhaustion. Using environmental and behavioral data points, the model predicts the likelihood of burnout to help HR departments intervene proactively.

Python Scikit-Learn Flask API Pandas Random Forest

Live Prediction Demo

Technical Overview

Dataset:

Trained on 22,000+ records of employee performance metrics and mental health surveys.

Model:

Gradient Boosting Regressor achieved an $R^2$ score of 0.91 during validation.

Workflow

  • Data Cleaning: Handled missing values in "Resource Allocation" using median imputation.
  • Feature Engineering: Created "Burnout Index" by combining physical fatigue and tenure.
  • Deployment: Model pickled and served via a Flask REST API with a responsive CSS frontend.
