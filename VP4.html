Woodland VP4 & Battery Man

LIVE SYSTEM

A dual-module database solution managing high-volume inventory for retail (Woodland VP4) and cataloging creative assets for a comic series (Battery Man). This project focuses on relational integrity and efficient query optimization.

MySQL PHP 8.x CRUD Architecture Responsive UI

Architecture

  • Normalized SQL schema (3NF) to minimize data redundancy.
  • Prepared statements in PHP to prevent SQL injection attacks.
  • Indexed product IDs for sub-millisecond query response.

Key Features

  • Real-time stock alerts for low-inventory "VP4" models.
  • Relational mapping between comic issues and characters.
  • Dynamic dashboard for "In-Stock" vs "Out-of-Stock" tracking.
