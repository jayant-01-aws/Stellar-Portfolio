Portfolio & Fibonacci Sphere

Live Project

A signature piece of creative engineering. This project demonstrates the intersection of mathematics and front-end art by mapping points onto a spherical surface using the Fibonacci Gold Ratio, rendered at a smooth 60fps using the HTML5 Canvas API.

HTML5 Canvas JavaScript ES6 Mathematical Modeling 3D Projection

The Mathematics

  • Golden Ratio ($\phi \approx 1.618$) used for point distribution.
  • Spherical-to-Cartesian coordinate transformation.
  • Dynamic perspective scaling for 3D depth.

UI/UX Features

  • Real-time rotation based on mouse/touch velocity.
  • Dynamic color shifting based on point depth ($Z$-axis).
  • Optimized `requestAnimationFrame` rendering loop.
